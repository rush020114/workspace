* 네이티브의 모든 컴포넌트는 
   display : flex
   flex-derection : column
   css 속성이 적용되어 있음

1. 네이티브는 프로젝트 실행 시 자동으로 시작하는 파일이 정해져 있음
    0순위 : _layout.jsx, 1순위 : index.jsx

2. 기본 컴포넌트
 0) 최상위 태그는 StatusBar와 겹침을 방지하기 위해 반드시 <SafeAreaView>를 사용 할 것!
 1) <View></View> -> <div> 와 동일한 역할
 2) <Text></Text> -> <p>와 비슷한 역할,  모든 글자는 반드시 <Text> 컴포넌트로 감쌀 것!
 3) <TextInput /> -> <input> 태그 역할
 4) <Button></Button> -> 버튼 생성. 사용하지 않음(안드, ios 디자인이 다르기 때문)
 5) <Pressable></Pressable> -> onClick 이벤트역할
      ex> 버튼 만드는 예제
             <Pressable onPress={() => { }}>
                  <Text>버튼</Text> 
             </Pressable>

3. TextInput 컴포넌트의 이벤트는 onChangeText를 사용!
     onChangeText 이벤트의 매개변수는 입력한 글자를 의미함.
