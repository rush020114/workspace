react 프로젝트 생성 명령어 : npm create vite@latest
생성한 프로젝트에 라이브러리 설치 명령어 : npm i
프로젝트 실행 명령어 : o + enter
프로젝트 재시작 명령어 : r + enter
프로젝트 종료 명령어 : q + enter

라이브러리 설치
npm install axios
npm install react-router-dom
npm install dayjs
 npm install @reduxjs/toolkit react-redux
npm install install jwt-decode
