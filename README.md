## 💡 배운 점

### 1. 반복 학습을 통한 기술 내재화
- 150일 연속 NOTION 작업일지 작성
- 같은 프로젝트를 3번 반복하며 핵심 개념 완전 습득
- 기능별 흐름과 구현 패턴을 몸으로 체화

### 2. 효율적인 코드 작성
- 중복 코드 50% 제거
- 변수명 통일로 80% 일관성 확보
- 문제 해결 능력 향상

### 3. 완성도 있는 프로젝트 추구
- 기본 실력을 아니라 소통 능력 중시
- 데이터베이스 설계 및 SQL 작성 능숙
- 알고리즘 문제 100점 달성
